VIDEO & AUDIO USING HTML

<!DOCTYPE html> <html> <head> <style> * { background-color:white; } </style> </head> <body> <video controls> <source src="video/face video.mp4"> </video> <audio controls> <source src="video/dhoni bgm.mp3"> </audio> <img src="images/dhoni-six.jpg"> </body> </html>

Output





The video tag is used to embed video files in a webpage.

The audio tag is used to play sound files.

The source tag specifies the media file path.

The controls attribute provides play and pause options.

The img tag is used to display images.
